# Starfruit
Starfruit is a fruit crop that grows from Starfruit Seeds after 13 days. It has the second-highest sell price of any crop in the game, with only the Sweet Gem Berry selling for more. One Starfruit can be purchased each year at the Luau for $3000.

## Tailoring

Starfruit can be used in the spool of the Sewing Machine to create the dyeable Genie Pants. It can be used in dyeing, serving as a yellow dye at the dye pots, located in Emily's and Haley's house.

## Quests

* Kent requests a Starfruit by mail on the 15th of Summer (year 2) in the "A Soldier's Star" Quest. The reward is $500 and 1 Friendship heart.

## History

Introduced in version 1.0.
Reduced value to $750, down from $800, in version 1.1.
Can now be used in Tailoring, in version 1.4.
Iridium quality is now possible when using Deluxe Fertilizer. Can now be bought at the Luau for $3000. Is now an option in the remixed Dye Bundle, in version 1.5.
